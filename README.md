# 🔥 TechVyana Live Input Tracker

A powerful Python-based **real-time keyboard monitoring system** with:

* 🖥️ Desktop GUI (Tkinter)
* 🌐 Web dashboard (Flask)
* 📱 Phone access over local network
* 📊 Typing stats (WPM, key count)

---

## 🚀 Features

* ⌨️ Real-time keyboard tracking
* 🌐 Live web view (accessible from phone)
* 📊 Typing analytics (WPM, key count)
* ⚡ Auto server start
* 🧠 Caps Lock & Shift handling
* 📡 Works over local network (WiFi)

---

## 🛠️ Tech Stack

* Python
* Tkinter
* Flask
* Pynput
* Requests

---

## 📦 Installation

```bash
pip install pynput flask requests
```

---

## ▶️ Run

```bash
python main.py
```

---

## 🌐 Access

After running, you will see:

* PC: http://127.0.0.1:5000
* Phone: http://YOUR-IP:5000

👉 Make sure:

* Same WiFi network
* Firewall allows Python
